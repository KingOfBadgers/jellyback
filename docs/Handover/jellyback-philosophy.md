# JellyBack Design Philosophy

JellyBack is not an automated poster generator.

JellyBack is a composition engine that gives users creative control over media presentation.

Core principles:

1. User choice always overrides automation.

2. The system never chooses layouts on behalf of the user.

3. Variants should only be hidden when impossible to build.

4. Rendering must be deterministic.

5. All compositions must be built before rendering.

6. Renderer components must never contain layout intelligence.

7. CSS variants require optional background treatments.

8. Asset availability controls variant visibility.

9. The system exists to expose creative options, not optimize decisions.

10. JellyBack behaves like design software, not recommendation software.
